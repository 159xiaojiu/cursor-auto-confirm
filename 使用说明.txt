========================================
  Cursor 自动确认助手  v0.1
  一键自动点击 Accept / Run / Continue
========================================

【适用环境】
  - Windows 10 / 11 (64位)
  - 已安装 Cursor 编辑器
  - 无需安装 Python，解压即用

【第一次使用 - 2 步】

  1. 解压本文件夹到任意位置（建议英文路径）

  2. 双击「创建桌面快捷方式.bat」
     → 桌面出现「Cursor自动确认」图标

【日常使用 — 像普通软件一样】

  双击桌面图标 → 弹出小窗口（任务栏可见）：
    ▶ 启动  → 开始自动点 Accept / Run / Continue
    ■ 停止  → 暂停自动点击
    退出程序 → 完全关闭

  不用进文件夹、不用点 bat 文件。

【分享给朋友】

  发送 release 里的 zip 压缩包即可。
  对方解压 → 创建桌面快捷方式 → 双击图标使用。

【安全提示】

  - 对 Run/终端命令会先检查危险命令（如 rm -rf），命中则拦截
  - 日志：logs\autopilot.log

【常见问题】

  Q: Windows 提示「未知发布者」？
  A: 点「更多信息」→「仍要运行」即可。

  Q: 多个 Cursor 项目能同时照看吗？
  A: 可以，默认扫描所有打开的 Cursor 窗口。

  Q: 没有自动点击？
  A: 确认窗口里已点「▶ 启动」；Cursor 需打开且 Agent 在运行。

【高级】
